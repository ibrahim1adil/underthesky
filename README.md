# Under the Sky

A personal photo journal built for IT-WEB 112 (Personal Website Creation project).
Night-sky, hiking, and canyon photography from the Pacific Northwest and the Southwest.

## Pages
- `index.html` — Home (featured hero + gallery links)
- `stars.html` — Stars gallery
- `hikes.html` — Hikes gallery
- `canyons.html` — Canyons gallery
- `about.html` — About the site and gear
- `contact.html` — Suggest-a-spot form
- `style-guide.html` — Project style guide (deliverable)

## Structure
```
under-the-sky/
├── index.html  stars.html  hikes.html  canyons.html  about.html  contact.html
├── style-guide.html
├── css/
│   ├── format.css      (colors + typography)
│   ├── layout.css      (structure, grids, responsive)
│   ├── effects.css     (hover, transitions, animation, lightbox)
│   └── styleguide.css  (style-guide page only)
├── js/
│   └── main.js         (contact-form thank-you message)
└── images/             (all original photos, organized by category)
```

## Notes
- All HTML and CSS validate with zero errors on the W3C validators.

## Publishing
To get a live link, push this folder to a GitHub repo and turn on
GitHub Pages (Settings → Pages → deploy from the main branch). The site
needs no build step — it's plain HTML/CSS/JS.
